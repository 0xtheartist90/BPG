# Epic 4: Agenda Events Management (Admin)

> CRUD interface for managing multilingual agenda events with recurrence support.

## Context

The site currently has 4 recurring event templates (Kunst in Gein, Kinderkledingpunt, Bewegingsexpressie, Repair Cafe) and 2 fixed events, all generated by code in `src/lib/translations.ts`. The client needs to manage events themselves, including creating recurring events without manual date entry.

## Technical Decisions

- **Recurrence model**: Option A — expand on save. Template row + individual instance rows generated immediately.
- **Recurrence patterns**: Weekly, Biweekly, Monthly (3 options only — sufficient for neighborhood platform)
- **Instance management**: Individual dates can be cancelled (published=false) or restored without affecting the series
- **Edit scope**: Editing a recurring template offers choice: "only template" or "all future events"
- **Images**: Reference paths for v1 (no upload)
